Design Tokens

Color palette, typography, spacing system

Primary Color (Berserker Red)
#ffb3b1 — Critical actions, achievements, high priority
Secondary Color (Primal Orange)
#ffbd5c — Energy states, active timers, metabolic data
Background
#1e0f0f — Deep onyx, reduces ocular strain
Surface
#1a1a1a — Cards and containers with glassmorphism
Typography - Headlines
Anton, 28-48px, uppercase, aggressive cinematic feel
Typography - Body
Inter, 14-16px, clear readable in gym environment
Typography - Technical
JetBrains Mono, 12px, data labels and technical info
Spacing Base Unit
4px — Foundation for all spacing and layout
Border Radius
0px (Sharp) — Brutalist aesthetic, except progress rings
Glassmorphism
20px backdrop blur, 1px borders, 10-20% opacity overlays
