# BerserkerCut - App Design System & Prototypes

## 🎨 Design System
Voir `DESIGN.md` pour la specification complete du design system.

### Caracteristiques principales
- **Aesthetic:** Brutalism + Glassmorphism industriel
- **Couleurs:** Palette onyx profond avec accents Berserker Red (#ffb3b1) et Primal Orange (#ffbd5c)
- **Typographie:** Anton (headlines), Inter (body), JetBrains Mono (technical labels)
- **Composants:** Sharp corners (0px), glass panels avec backdrop blur, progress rings épais
- **Spacing:** Base 4px, fluid grid 4 colonnes (mobile) / 12 colonnes (desktop)

## 📱 Interactive Prototypes

Voir le dossier `stitch_berserkercut_ai_nutrition_training_ecosystem/` pour les prototypes HTML interactifs:
- `daily_pilot_dashboard/` - Tableau de bord principal avec adherence score
- `nutrition_tracking/` - Suivi nutrition et macros
- `training_plan/` - Planning entrainement hebdomadaire
- `onboarding_*` - Flux d'inscription (identite, sante, entrainement, supplements)
- `system_settings/` - Parametres app (theme, notifications, confidentialite)
- `weekly_agenda_reminders/` - Agenda et rappels
- `auth_consent/` - Authentification et consentement cloud

## 🚀 Convention Captures Ecrans

Pour les captures d'ecran reelles (screenshots app iOS):
- Placer dans un sous-dossier `screenshots/` numerote
- Format: `01-auth-login.png`, `02-onboarding-health.png`, etc.
- Inclure variantes dark + light si possible
- Aucune donnee personnelle reelle
