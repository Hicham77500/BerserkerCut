## 🎨 Points clés du Design System

- **Industrial Technical:** Brutalism + Glassmorphism
- **Palette:** Charcoal profond (#1e0f0f) + Berserker Red (#ffb3b1) + Primal Orange (#ffbd5c)
- **Typographie:** Anton (impact), Inter (lisibilite), JetBrains Mono (donnees)
- **Composants:** Sharp corners, glass panels, progress rings 8-12px
- **Interaction:** Responsive mobile-first, tab bar frosted glass en bas
